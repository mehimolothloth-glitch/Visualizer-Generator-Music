# Visualizer & Generator Music

# Sebuah projek yang saya kembangkan dengan bantuan AI DeepSeek. Projek dengan beberapa fitur untuk kenyamanan memutar musik dari playlist file musik anda.
# Pengembangan bersama meliputi:
* **Web Audio API & Visualisasi:** Membantu integrasi API Audio agar partikel canvas bergerak dinamis merespons melodi musik.
* **Audio Synthetis:** Membantu pembuatan algoritma *Noise Generator* untuk menciptakan suara hujan yang natural.
* **Debugging & Optimization:** Membantu pemecahan masalah (*troubleshooting*) kode dan optimalisasi performa aplikasi.

# Fitur:
 - Suara hujan (noise)
 - Lo-fi
 - upload file musik
 - Tombol play & pause

# Fitur unik: Partikel Partikel yang bisa bergerak mengikuti melodi musik.

# Bahasa pemrograman: HTML, CSS, Javascript.

Uji tes: Mampu berjalan normal 88% browser & melalui pengujian ketat terhadap keamanan API dan privasi pengguna.
